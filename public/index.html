Welcome To The Store API

Endpoints Should Be As Follows

1- /createStore --> Posts a new store to the database.

2- /viewStores --> Returns a list of stores names that are available in your database.

3- /viewStores/{storeID} --> Returns the details of the store that has an id equal to storeID.

4 - /updateStore/{storeID} --> Updates the information of the store that has an id equal to storeID through a PUT method.
